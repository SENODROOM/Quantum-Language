We welcome contributions through GitHub forks! Here's how you can contribute:

#### 1. Fork the Repository
1. Visit [Quantum Language on GitHub](https://github.com/SENODROOM/Quantum-Language)
2. Click the **"Fork"** button in the top-right corner
3. This creates a personal copy of the repository

#### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/Quantum-Language.git
cd Quantum-Language
```

#### 3. Create a Feature Branch
```bash
git checkout -b feature/your-feature-name
# or
git checkout -b fix/your-bug-fix
```

#### 4. Make Your Changes
- Code changes
- Documentation updates
- Bug fixes
- New features
- Test improvements

#### 5. Test Your Changes
```bash
# Build the project
.\build.bat

# Test the compiler
.\quantum.exe --version
.\quantum.exe examples\hello.sa
```

#### 6. Commit Your Changes
```bash
git add .
git commit -m "feat: add your feature description"
```

#### 7. Push to Your Fork
```bash
git push origin feature/your-feature-name
```

#### 8. Create a Pull Request
1. Visit your fork on GitHub
2. Click the **"New Pull Request"** button
3. Provide a clear title and description
4. Wait for review and feedback
